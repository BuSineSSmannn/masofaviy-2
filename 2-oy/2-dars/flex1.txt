flex-direction	Определяет направление гибких элементов внутри гибкого контейнера


flex-direction: row;

flex-direction: row-reverse;

flex-direction: column;

flex-direction: column-reverse;



justify-content	Выравнивание элементов по горизонтали по горизонтали, когда элементы не используют все доступное пространство на главной оси




justify-content: center;     /* Выравнивание элементов по центру */
justify-content: flex-start; /* Выравнивание элементов с самого начала */
justify-content: flex-end;   /* Выравнивание элементов с самого конца */


justify-content: space-between; /* Равномерно распределяет все элементы по ширине flex-блока.
                                   Первый элемент вначале, последний в конце */
justify-content: space-around;  /* Равномерно распределяет все элементы по ширине flex-блока.
                                   Все элементы имеют полуразмерное пространство

                                   с обоих концов */
justify-content: space-evenly;  /* Равномерно распределяет все элементы по ширине flex-блока.
                                   Все элементы имеют равное пространство вокруг */
justify-content: stretch;       /* Равномерно распределяет все элементы по ширине flex-блока.
                                   Все элементы имеют "авто-размер", чтобы соответствовать

                                   контейнеру */





align-items	 Вертикальное выравнивание гибких элементов, когда элементы не используют все доступное пространство на поперечной оси



align-items: flex-start;

align-items: flex-end;

/* Центрировать элементы в поперечной оси */
align-items: center;



/* Растянуть предметы, чтобы соответствовать */
align-items: stretch;



flex-wrap Указывает, следует ли переносить элементы flex или нет, если для них недостаточно места на одной линии flex

order	Определяет порядок гибкого элемента относительно остальных гибких элементов в том же контейнере

align-self	Используется на гибких предметах. Переопределяет свойство align-items контейнера

 flex-grow Свойство указывает, насколько гибкий элемент будет расти относительно остальных гибких элементов.
 
flex-shrink указывает, насколько изгибаемый элемент будет сжиматься относительно остальных элементов flex.

 flex-bases указывает начальную длину элемента flex.



flex	Сокращенное свойство для свойств flex-grow flex-shrink и flex-based
flex: 1 1 300px;
